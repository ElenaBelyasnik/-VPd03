Приложение для получения и отображения погоды по названию города или координатам.

## Файлы проекта

- `weather_app.py` — основной модуль для получения погоды
- `weather_info.py` — модуль для красивого отображения погоды в консоли
- `weather.json` — файл с данными о погоде (создаётся автоматически)
- `weather_cache.json` — кэш данных (создаётся автоматически)
- `.env` — файл с API ключом (не коммитится!)
- `requirements.txt` — зависимости проекта

## Установка

## Использование

Запустите приложение в интерактивном режиме:
```powershell
python weather_app.py
```

Меню приложения:
1. **По названию города (текущая погода)** — введите название города на русском языке
2. **По координатам (текущая погода)** — введите широту и долготу
3. **По названию города на 5 дней** — прогноз с шагом 3 часа
4. **По координатам на 5 дней** — прогноз с шагом 3 часа
5. **Расширенный режим по городу** — с данными о качестве воздуха
6. **Расширенный режим по координатам** — с данными о качестве воздуха
7. **Выход** — завершить работу

### Дополнительные команды:
```powershell
# Получить погоду для конкретного города из командной строки
python weather_app.py --city Москва

# Прогноз на 5 дней
python weather_app.py --city Москва --days 5

# Прогноз по координатам
python weather_app.py --coords 55.75 37.61 --days 3

# Расширенный режим (с качеством воздуха)
python weather_app.py --city Москва --extended

# Прочитать погоду из файла
python weather_app.py --read
python weather_info.py
```

## Особенности

### Кэширование
- Данные автоматически сохраняются в `weather_cache.json`
- При сетевых ошибках, если кэш не старше 3 часов, данные показываются из кэша
- Кэш проверяет совпадение города и количества дней

### Повторные запросы (Retry)
- При ошибке 429 (слишком много запросов) или сетевых ошибках — до 3 повторов
- Экспоненциальная задержка между повторами: 1с, 2с, 4с

### Поддерживаемые форматы ввода
- Города на русском языке: Москва, Санкт-Петербург, Казань и др.
- Координаты: широта и долгота (например, 55.75, 37.61)
- Количество дней: от 1 до 7

## Функции

### weather_app.py
- `get_coordinates(city)` — получает координаты города через Nominatim API
- `get_weather_by_coordinates(lat, lon, days, extended)` — получает погоду от Yandex Weather API
- `get_current_weather(city, lat, lon, days, extended)` — универсальный запрос погоды
- `save_weather_to_file(weather_data, filename)` — сохраняет погоду в JSON файл
- `save_to_cache(weather_data, city, lat, lon, days)` — сохраняет данные в кэш
- `is_cache_valid(city, lat, lon, days)` — проверяет валидность кэша

### weather_info.py
- `read_weather_from_file(filename)` — читает погоду из JSON файла
- `print_weather_summary(weather)` — отображает текущую погоду с прогнозом на день
- `print_forecast_table(weather)` — отображает прогноз на несколько дней в виде таблицы
- `print_extended_weather(weather)` — отображает расширенную погоду с качеством воздуха
